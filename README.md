# Muhammad Usman Ghani Portfolio

Static portfolio website for Muhammad Usman Ghani, Senior Planning Engineer and Project Controls Specialist.

## GitHub Pages

This site is built with plain HTML, CSS, and JavaScript. To publish it:

1. Push the repository to GitHub.
2. Open the repository settings on GitHub.
3. Go to **Pages**.
4. Select **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)` folder.
6. Save.

The site will be available at:

`https://uxmanghani-108.github.io/usmanghani/`
